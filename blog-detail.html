01 วิธีการติดตั้ง Python IDLE

การติดตั้ง IDLE ของ Python จาก python.org

27 เมษายน 2021 | 1,000

การติดตั้ง python IDLE 

เราต้องไปที่ เว็บไซต์ https://www.python.org/


เพื่อดาวน์โหลดตัวติดตั้ง กันนะครับ

ในหน้าการดาวน์โหลดจะ มี Package ของ Python ให้เราเลือกติดตั้ง เวอร์ชันล่าสุด คือ Python 3.9



ให้เลื่อนลงมา  แล้วเลือกดาวน์โหลดเวอร์ชันตามที่ต้องการ  ในบทความนี้จะ ติดตั้งเวอร์ชัน 3.9.4 


ต่อไปเลือกโหลดไฟล์ที่เหมาะสมกับระบบปฏิบัติการ ในบทความนี้จะติดตั้งตัว Windows installer (64-bit)
